### Opening a new issue

Any issue that fails to follow these instructions will be automatically closed and marked as invalid:

For any issue created it must be in English only.  
For any issue created it must only refer to one issue only.  
Do not include more than one subject (problem, request or both) in one issue.  

#### Bug/problem report:
 - Follow the instructions for reporting a bug here: https://github.com/NovaViper/Console-Master/wiki/Report-A-Bug

#### Feature requests:
 - Feature requests will be automatically be labeled as "Feature" for easier access and better organization
 - If a feature request is considered to be implemented it will be re-opened for the duration of the implementation

### Contributing to source code

While help is appreciated, source code submissions will be checked and decided on. When doing submissions, please use forks and create pull requests.
